# ArrayBuffer

> ArrayBuffer는 브라우저와 Node.js 모두에서 사용할 수 있는 원시 바이너리 데이터 버퍼
>
> > typed arrays나 DataView를 통해서 접근

## ArrayBufferView

> ArrayBuffer에 접근하기 위한 객체

## DataView

> ArrayBufferView의 한종류

## AudioBuffer

> 오디오 용도

## SourceBuffer

> 미디어 용도

## TypedArrays

> ArrayBufferView의 한종류
>
> > for reading and writing raw binary data in memory buffers.
