# loader & transpiler

> 둘 다 소스 코드를 변환하는데 사용되는 도구들

## loader

> 웹팩과 같은 번들러에서 사용되는 개념
>
> > 로더는 다른 언어를 자바스크립트 문법으로 변환해주거나 이미지를 데이터 URL 형식의 문자열로 반환 등의 역할을 한다.
> >
> > > ts를 js로 바꾸거나 scss를 css로 바꾸는 것
> > >
> > > > tsc같은 컴파일러(트랜스파일러)와 웹팩같은 번들러를 이어주는 역할

## transpiler

> 바벨과 같은 코드 변환 도구
>
> > 호환되지 않는 문법을 바꿔주는 역할
