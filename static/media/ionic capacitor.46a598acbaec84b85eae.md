# ionic Capacitor

> Cordova를 사용했으나 최신 버전부터는 Capacitor 사용 (deprecated)

## install

```sh
npm i @capacitor/core
npm i -D @capacitor/cli
```

## init

```sh
npx cap init
```

## android ios

```sh
npm i @capacitor/android @capacitor/ios

npx cap add android
npx cap add ios
```

## sync

```sh
npx cap sync
```

## command

```sh
npx cap [명령어]
```

## 앱 안드로이드 스튜디오에서 열기

```sh
npx cap open android
```
