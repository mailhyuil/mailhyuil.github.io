# 코드 종류 (byte, native, text, binary)

## 바이트 코드 vs 네이티브 코드

```txt
바이트 코드
- 인터프리터가 해석할 수 있는 중간 코드
- 플랫폼 독립적
- 자바, 파이썬, JS 등에서 사용
- VM이 실행

네이티브 코드
- CPU가 직접 실행할 수 있는 코드
- 플랫폼 종속적
- C, C++, Go 등에서 사용
- OS가 실행
```

## 텍스트 코드 vs 바이너리 코드

```txt
텍스트 코드
- 사람이 읽을 수 있는 코드
- 가독성이 좋음
- 실행 불가능
- 예: HTML, XML, JSON 등

바이너리 코드
- 사람이 읽을 수 없는 코드
- 가독성이 나쁨
- 실행 가능
- 예: 이미지, 오디오, 비디오, 바이트 코드, 네이티브 코드 등
```
