# 페이징 처리

```
num_page_no = 2 // 페이지번호
num_page_size = 100 // 한페이지출력개수(사이즈)

num_start_row = ((num_page_no-1) * num_page_size) + 1  // 출력 페이지 시작 행 (ex. 101)
num_end_row   = (num_page_no * num_page_size)          // 출력 페이지 마지막 행 (ex. 200)
```


- 오라클
```
SELECT *
FROM
    (
        SELECT /*+ INDEX(T1 PK1) */
            ROWNUM AS RNUM, T1.*
        FROM
            tbl_bbs T1
        WHERE
            ROWNUM <= 200
    )
WHERE 101 <= RNUM;
```
