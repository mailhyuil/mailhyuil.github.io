# linux base log

## /var/log/messages

> 시스템에서 발생하는 표준 메시지가 기록되는 파일
>
> > root 권한으로만 접근 가능

## /var/log/secure

> 인증과 관련된 로그가 기록
>
> > 로그인, tcp_wrappers, xinetd 관련 로그

## /var/log/lastlog

> 마지막으로 로그인한 사용자의 정보가 기록

## /var/log/dmesg

> 부팅 기록

## /var/log/wtmp

> 로그인 성공 기록

## /var/log/btmp

> 로그인 실패 기록

## /var/log/sulog

> su 명령어 사용 기록

## /var/log/boot.log

> 부팅시 파일 시스템 마운트, 네트워크 설정, 서비스 시작 등 부팅과 관련된 로그

## /var/log/cron

> cron 관련 로그

## /var/log/maillog

> 메일 관련 로그

## /var/log/xferlog

> ftp 로그

## /var/log/nginx/access.log

> nginx 접속 로그

## /var/log/nginx/error.log

> nginx 에러 로그
