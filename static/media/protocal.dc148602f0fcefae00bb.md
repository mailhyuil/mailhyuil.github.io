# 프로토콜

> TCP, HTTP, HTTPS, MQTT, AMQP(RabbitMQ), Websocket(ws, wss), RESP protocol(redis), Postgres protocol는 프로토콜
>
> > gRPC는 프레임워크 (RPC : 원격 프로시저(메소드) 호출) (프로시저는 함수나 메소드와 비슷한 용어)
> >
> > > RPC는 기존에 SOAP 프로토콜을 사용해서 구현하기도 했다

# 바이너리 기반 프로토콜 vs 텍스트 기반 프로토콜

> 바이너리 기반 : tcp, mqtt, amqp...
>
> > 텍스트 기반 : http, websocket...
> >
> > > 바이너리 기반이 더 빠르다
