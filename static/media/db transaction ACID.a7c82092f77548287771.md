```sh
Atomicity # 원자성

```

## 무결성

```sh
*Integrity # 무결성 /// 데이터베이스에서 저장된 데이터가 정확하고 일관된 상태를 유지하는 것을 의미
           # 관계형 데이터베이스의 가장 큰 목표는 "데이터 무결성을 높이는 것"

Entity Integrity (개체 무결성)
Referential Integrity (참조 무결성)
Domain Integrity (도메인 무결성)
Transaction Integrity (트랜잭션 무결성)
```
