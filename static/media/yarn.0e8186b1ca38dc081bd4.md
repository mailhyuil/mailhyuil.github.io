# yarn

## install

```sh
yarn
# or
yarn install
```

## add

> 패키지 추가

```sh
yarn add <package>
```

## remove

> 패키지 삭제

```sh
yarn remove <package>
```

## upgrade

> 패키지를 전부 최신 버전으로 업그레이드

```sh
yarn upgrade
yarn upgrade <package>
# 특정 패키지를 특정 버전으로 업그레이드
yarn upgrade <package>@<version>
# 목록들 중에서 원하는 패키지만 interactive하게 최신버전으로 업그레이드하는 ui
yarn upgrade-interactive
```
