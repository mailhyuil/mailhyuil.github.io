# 리눅스 셸 변수

```sh
$$ : 쉘의 프로세스 번호 (pid)
$0 : 쉘 스크립트 이름
$1~$9 : 명령줄 인수
$* : 모든 명령줄 인수 리스트
$# : 인수의 개수
$? : 최근 실행한 명령어의 종료 값

$HOME : 홈 디렉토리
$PATH : 실행 파일을 찾는 경로
$PS1 : 첫번째 프롬프트 문자열
$PS2 : 두번째 프롬프트 문자열
$IFS : 필드 구분자
$TERM : 터미널 타입
$SHELL : 로그인 쉘
$USER : 사용자 이름
$UID : 사용자 ID
$GROUPS : 사용자 그룹
$HOSTNAME : 호스트 이름
$PWD : 현재 디렉토리
$OSTYPE : 운영체제 종류
$MACHTYPE : 시스템 종류
```
