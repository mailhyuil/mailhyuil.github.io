# TestBed

> The TestBed creates a dynamically-constructed Angular test module that emulates an Angular @NgModule.
>
> > TestBedлҠ” @NgModuleмқ„ нқүлӮҙлӮҙлҠ” лҸҷм ҒмңјлЎң кө¬м„ұлҗң Angular н…ҢмҠӨнҠё лӘЁл“Ҳмқ„ мғқм„ұн•©лӢҲлӢӨ.

```ts
TestBed.inject(); // TestModuleмқҳ providersм—җ л“ұлЎқлҗң providersлҘј мЈјмһ…
TestBed.createComponent(); // TestModuleмқҳ importsм—җ мһ„нҸ¬нҠёлҗң ComponentлҘј мғқм„ұ
```
