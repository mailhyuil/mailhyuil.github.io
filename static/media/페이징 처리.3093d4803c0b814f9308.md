# 페이징 처리 (pagination)
- 총 페이지 수
```
totalPageNum = Math.ceil(totalSize/PAGE_NUM_SIZE);

/*OR*/

long totalPageNum = totalSize/PAGE_NUM_SIZE;

if(totalSize % PAGE_NUM_SIZE != 0) {
    totalPageNum++;
}
```

- 첫번째 페이지 번호와 마지막 페이지 번호 구하기
```
 // 현재 페이지 번호
long currentPageNum = ?;

 // 한 페이지 출력 개수(사이즈)
long PAGE_NUM_SIZE = 100;

//첫번째 페이지 번호 구하기
long firstPageNum = (currentPageNum  / PAGE_NUM_SIZE) * PAGE_NUM_SIZE;
long firstPageNum = currentPageNum - (currentPageNum % PAGE_NUM_SIZE);

long firstPageNum = currentPageNum - (currentPage-1) % PAGE_NUM_SIZE; // newlec

// 마지막 페이지 번호 구하기
long lastPageNum = startPageNo + PAGE_NUM_SIZE - 1;
long lastPageNum = currentPageNum - (currentPageNum % PAGE_NUM_SIZE) + PAGE_NUM_SIZE;
long lastPageNum = currentPageNum - (currentPageNum % PAGE_NUM_SIZE) + PAGE_NUM_SIZE;
```

- 첫번째 페이지와 마지막 페이지 에러 처리
```
firstPageNum = firstPageNum < 1 ? 1 : firstPageNum;
lastPageNum = lastPageNum > totalPageNum ? totalPageNum : lastPageNum;
```
