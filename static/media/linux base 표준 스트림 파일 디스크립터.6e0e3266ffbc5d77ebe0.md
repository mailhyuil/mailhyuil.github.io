# 표준 스트림

> stdin, stdout, stderr
>
> > 리눅스에서는 모든 것이 파일이다
> >
> > > 프로세스가 시작되면 3개의 파일이 자동으로 열린다 (stdin, stdout, stderr)
> > >
> > > > 만약 파일을 open한다면 4번째 파일 테이블에 추가된다.

## 파일 디스크립터

```sh
stdin = 0 # 표준 입력 스트림 // 표준장치 (키보드)
stdout = 1 # 표준 출력 스트림 // 표준장치 (모니터)
stderr = 2 # 표준 에러 스트림 // 표준장치 (모니터)

## 입력을 다른 기기로 바꾸고 싶을 때
## 0번 파일 디스크립터를 사용하고 싶은 파일로 바꾼다.
```
