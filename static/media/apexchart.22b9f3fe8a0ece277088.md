# apexChart

## install

```
npm install --save apexcharts
npm install --save vue3-apexcharts
```

## plugin

```
import VueApexCharts from "vue3-apexcharts";
defineNuxtPlugin((nuxtApp) => {
	nuxtApp.vueApp.use(VueApexCharts);
});
```

## 사용

```
<apexcharts
    type=""
    series=""
    width=""
    height=""
    options=""
></apexcharts>
```
