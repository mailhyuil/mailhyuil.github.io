# nextui

> next에서 사용할 수 있는 UI 라이브러리
>
> > react18, framer-motion, tailwindcss을 기반으로 한다.

## install

```sh
# cli 설치
npm i -D nextui-cli

npx nextui --add
```

## usage

```tsx
import React from "react";
import { Button } from "@nextui-org/react";

export default function App() {
  return <Button color="primary">Button</Button>;
}
```
