# npm ci

> npm ci 명령어는 주로 프로젝트의 종속성(dependency)을 설치하기 위해 사용됩니다.
>
> > npm install과 유사한 기능을 수행하지만, package-lock.json 파일이 존재할 경우에만 동작합니다.
> >
> > > package-lock.json 파일은 프로젝트의 종속성 트리(dependency tree)를 정확하게 재현하기 위해 사용되는 파일입니다.
> > >
> > > > 주로 CI/CD(Continuous Integration/Continuous Deployment) 시스템이나 테스트 환경에서 사용되며, 종속성을 정확하게 설치하고 일관성을 유지하는 데 도움을 줍니다.

```sh
npm ci
```
