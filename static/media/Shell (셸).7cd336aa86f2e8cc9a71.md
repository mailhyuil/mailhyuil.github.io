# Shell (셸)

> 리눅스 시스템에서 명령어를 입력하는 장소
>
> > /bin/bash에 저장된 명령어를 실행 (echo $SHELL -> /bin/bash)
> >
> > > 셸 내장명령어도 있다 (type 명령어로 확인 가능)

## SHELL 종류

```
sh = bourn shell
bash = bourn again shell
ash = 암키스트 shell
csh = c shell
zsh = z shell
tcsh = tee-see shell
cmd
```

## 콘솔

> 로컬 장치에 명령을 입력할 수 있는 소프트웨어

## 터미널

> 로컬 및 원격으로 명령을 입력할 수 있는 소프트웨어
