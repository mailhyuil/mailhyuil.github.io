# ionic framework

## install

```sh
npm install -g @ionic/cli
```

## create project

```sh
ionic start
```

## UI Components

> 미리 컴포넌트로 제공되는 UI를 사용
>
> > ion-\* 으로 되어있는 태그 사용

```html
<ion-router-outlet></ion-router-outlet>

<ion-header></ion-header>

<ion-content></ion-content>

<ion-footer></ion-footer>
```
