# 클로저

> 내부 함수가 종료된 외부 함수의 변수를 참조하는 메커니즘
>
> > 자신이 정의된 스코프(렉시컬 스코프)에 정의된 값들을 캡처한다.

## example

```
const hello = (name) => {
  let _name = name;
  return () => { // 리턴하는 함수는 _name을 기억한다.
    console.log("Hello, " + _name);
  };
};

let hello1 = hello("sb"); // 리턴받은 함수를 변수에 저장해도 _name의 값은 남아있다.

hello1(); // Hello, sb
```
