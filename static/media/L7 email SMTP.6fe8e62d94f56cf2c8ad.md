# email SMTP (Simple Mail Transfer Protocol)

> 받는 사람의 SMTP 서버와 보내는 사람의 SMTP 서버가 통신하여 메일을 전송하는 프로토콜
>
> > SMTP 서버는 naver, gmail 등이 제공한다.
> >
> > > SMTP 서버가 이메일을 받으면 POP3, IMAP4 등의 프로토콜을 사용하여 사용자의 메일 서버로 전송한다.
> > >
> > > > 다른 TCP/IP 프로토콜과는 다르게 이메일 프로토콜은 ip와 port가 아닌 domain을 사용한다. (기기가 아닌 사용자 중심이기 때문에)
