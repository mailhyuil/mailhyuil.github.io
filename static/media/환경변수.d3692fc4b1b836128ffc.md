# 환경변수
- 특정 디렉터리 경로, 실행파일 또는 명령 등에 직접 접근하지 않고 사용할 수 있도록 환경변수 지정
- 생산성과 편의성을 높일 수 있음

## 기본 환경변수 목록
1. HOME: 사용자의 홈 디렉토리
2. PATH: 실행파일을 찾는 경로
3. LANG: 프로그램 사용 시 기본 지원되는 언어
4. USER: 사용자의 이름
5. PS1: 명령 프롬프트 변수
> 확인방법 : echo $HOME


- `환경변수키=값` 명령어를 쓰면 `지역환경변수` 값을 설정할 수 있다. (로그아웃시 값이 사라짐)
- `export 환경변수키=값` 명령어를 쓰면 `전역환경변수` 값을 설정할 수 있다. (로그아웃시 값이 사라짐)
> export JAVA_HOME=/usr/java/jdk1.6.0_45

- 환경변수 해제
> unset 환경변수키

- 사용자 세션이 열릴때마다 export명령어 수행을 위해 `~/.bash_profile` 설정파일 수정 (영구 적용)
- 사용자 세션이 열릴때마다 export명령어 수행을 위해 `/etc/profile` 설정파일 수정 (영구 적용)
- `/etc/profile.d`에 .sh 파일을 생성하여 적용 (OS 업데이트를 해도 환경변수 유지됨)
> `/tc/bash/bashrc`파일을 열어서 맨 마지막 줄에 적용하고자 하는 export 명령어 설정
> 특정 사용자에게만 적용하고 싶은 경우 `~/.bashrc` 파일을 수정
> `source ~/.bash_profile` 또는 `source /etc/profile`를 입력해 변경 내용 적용
- 환경변수 확인
```bash
# 전체 확인
env
printenv
# 특정 환경변수 확인
env | grep 환경변수키
```
