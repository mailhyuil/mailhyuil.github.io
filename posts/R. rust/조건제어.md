# 조건 제어

## if

```
if number < 5 {
    ...
} else if number < 3 {
    ...
} else {
    ...
}
```

### let 구문에서도 사용 가능

```
let condition = true;
let number = if condition {
    5
} else {
    1
}
```

## loop

> 루프를 중지하라고 명시하지 않는 한 계속 반복
>
> > break를 사용하면 빠져나옴

```
let mut counter = 0;

loop {
    counter += 1;
    if counter == 10 {
        break counter * 2;
    }
}
```

## while

```
while number != 0 {
    ...
}
```

## for

```
let a = [1,2,3,4,5];

for e in a.iter(){
    ...
}
```

```
for number in (1..4).rev(){
    ...
}
```
