# 내장모듈

## path
> 폴더와 파일의 경로를 쉽게 조작할 수 있도록 도와주는 모듈
## url
> 인터넷 주소를 쉽게 조작할 수 있도록 도와주는 모듈
```js
url.parse() //주소를 분해
url.format() // 분해된 url개체를 다시 조립
```
## os
> 운영체제
```
os.hostname()
os.freemem()
os.totalmem()
os.cpus()
os.networkInterfaces()
```
## crypto
> 암호화를 도와주는 모듈
```js
crypto.createHash().update().digest()
```
## fs
> 파일 입출력
>> 동기, 비동기 모두 지원
```js
fs.mkdir()
fs.mkdirSync()
fs.rmdir()
fs.rmdirSync()
fs.writeFile()
fs.writeFileSync()
fs.appendFile()
fs.appendFileSync()
fs.readFile()
fs.readFileSync()
fs.stat()
fs.statSync()
```
