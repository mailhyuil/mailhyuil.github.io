# ionic framework

## install

```
npm install -g @ionic/cli
```

## create project

```
ionic start
```

## UI Components

> 미리 컴포넌트로 제공되는 UI를 사용
>
> > ion-\* 으로 되어있는 태그 사용

## CSS utils

### flex

> ion-grid 사용

```
<ion-grid>
  <ion-row>
    <ion-col class="ion-text-center">
      <ion-button>hi</ion-button>
    </ion-col>
  </ion-row>
</ion-grid>
```
