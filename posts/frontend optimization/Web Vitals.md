# Web Vitals

## LCP

> Largest Contentful Paint
>
> > 로딩 성능을 측정
> >
> > > 페이지가 처음 로드를 시작하고 가장 큰 이미지 또는 텍스트 블록이 화면에 표시되는 시점

### 개선 방법

> 서버 응답 시간 개선 (가까운 CDN 서버를 사용)
>
> > 리소스 요청 최적화
> >
> > > css 축소, 지연, 즉시 처리

## INP

> Interaction to Next Paint

### 개선 방법

> 사용하지 않는 자바스크립트 분할
>
> > 자바스크립트 실행 시간 단축
> >
> > > 웹 워커 사용

## CLS

> Cumulative Layout Shift
>
> > 사용자가 예상치 못한 레이아웃 변화를 경험하는 시점 (시각적 안정성)
> >
> > > 요소가 추가되면서 레이아웃이 변화되는 것

### 개선 방법

> 이미지 리사이징, 이미지 크기를 미리 설정
>
> > 광고를 위한 공간을 미리 확보
> >
> > > 동적 요소를 미리 대응
> > >
> > > > FOUT/FOIT 방지 (폰트가 나중에 적용되는 현상)
