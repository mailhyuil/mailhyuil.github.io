# figma design to code

figma api + nodejs 환경에서 llm을 통해서 figma lib으로 tailwindcss 스타일로 ui 코드를 그리고, 완성된 UI를 builder.io를 통해서 다시 react, angular... 의 코드로 변환하여 소스코드에 머지하는 방식

1. Figma plugin API + nodejs 환경에서 llm을 통해 figma UI 생성 (tailwindcss 스타일로)
2. 완성된 UI를 builder.io를 통해서 react, angular.. 코드로 변환
3. CI/CD 수행
