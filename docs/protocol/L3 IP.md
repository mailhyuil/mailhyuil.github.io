# IP

> ip address를 통해 호스트(컴퓨터)를 식별할 수 있다

## 메시지 중심 프로토콜

> (데이터의 시작과 끝의 경계가 명확하다)

## 데이터 단위 : 패킷

> IP에서 TCP로 데이터를 보내는 단위
>
> > Header section : 20 byte
> >
> > Data section : 0 ~ 65535 byte

```sh
[source IP address][-----------------------DATA-----------------------][destination IP address]
```
