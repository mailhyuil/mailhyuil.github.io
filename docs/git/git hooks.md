# git hooks

> .git/hooks/

## .git/hooks

```sh
# 파일명
pre-commit          # 커밋 전 실행
post-commit         # 커밋 후 실행
pre-push           # 푸시 전 실행
post-checkout      # 체크아웃 후 실행
commit-msg         # 커밋 메시지 검증
pre-receive        # 서버에서 푸시 받기 전
post-receive       # 서버에서 푸시 받은 후
update             # 브랜치 업데이트 시
```
