# 해시함수 - hash

## SHA-256

> 256비트 이진 해시 값을 반환하는 해시함수

## CRC32 / Jenkins Hash / FNV-1a (Fowler-Noll-Vo)

> 32비트 이진 해시 값을 반환하는 해시 함수

## CRC256

> 256비트 이진 해시 값을 반환하는 해시 함수

## MD5

> 보안위험으로 보안 용도로는 사용 x
>
> > 파일 무결성 검사 용도로 사용
> >
> > 무결성: 파일이 변조되지 않았음을 보장하는 것 (손상, 변경, 삭제 등)

## SHA-1

> 보안위험으로 보안 용도로는 사용 x
