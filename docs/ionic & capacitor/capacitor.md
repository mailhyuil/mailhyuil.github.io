# capacitor

> The Native Bridge for Cross-Platform Web Apps
>
> > Ionic은 기존에 Cordova를 사용했으나 최신 버전부터는 Capacitor 사용

## install

```sh
npm i -g @capacitor/cli

npm i @capacitor/core
```

## init

```sh
npx cap init
```

## android ios

```sh
npm i @capacitor/android @capacitor/ios

npx cap add android
npx cap add ios
```

## sync

```sh
npx cap sync
```

## command

```sh
npx cap [명령어]
```

## 앱 안드로이드 스튜디오에서 열기

```sh
npx cap open android
```
