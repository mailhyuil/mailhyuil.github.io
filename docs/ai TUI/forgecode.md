# forgecode

## usage

```sh
!cmd

# forge mode
# 권장: 프로젝트 루트에서 실행하여 전체 코드베이스에 액세스
# 기능 구현, 버그 수정, 파일 작성을 수행하는 실행 에이전트입니다.
forge --mode=forge

# muse mode
# 권장: 특정 폴더나 파일에서 실행하여 해당 영역에 집중
# 복잡한 아키텍처나 레거시 코드를 파악할 때 사용하는 분석 및 컨텍스트 이해 전용 에이전트입니다.
forge --mode=muse

# sage mode
# 권장: 프로젝트 루트에서 실행하여 전체 코드베이스에 액세스
# 프로젝트 전반에 걸친 고급 분석과 통찰을 제공하는 전문가 에이전트입니다. 아키텍처 리뷰, 코드 품질 평가, 리팩토링 제안 등에 적합합니다.
forge --mode=sage
```
