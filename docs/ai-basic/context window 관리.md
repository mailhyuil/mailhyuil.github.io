# ai context window 관리

## TopK 그대로 넣지 말고 “예산”으로 관리

TopK 그대로 넣지 말고 “예산”으로 관리

예: “컨텍스트 총 6,000 토큰까지만”

top chunk부터 넣되, 예산 넘으면 stop

## 중복 제거

같은 문서에서 연속 청크 3개 들어가면 거의 같은 말 반복임

문서별 최대 N개 제한: 예) doc당 max 2 chunks

## Chunk 압축(Compression)

rerank로 뽑힌 청크를 그대로 넣기 전에:

“질문과 관련된 문장만 추출” (rule 기반도 됨)

또는 작은 요약 모델로 “질문 관련 요약” 생성 (비용은 좀 듦)
