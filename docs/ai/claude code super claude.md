# claude code super claude

- `https://superclaude.org/`

## commands

```sh
# 개발
/sc:implement
/sc:build
/sc:design
# 분석
/sc:analyze
/sc:troubleshoot
/sc:explain
# 품질
/sc:improve
/sc:test
/sc:cleanup
# 기타
/sc:document
/sc:git
/sc:estimate
/sc:spawn
...
```

## options

```sh
--seq : sequential thinking
--persona-<name> : set the persona for the conversation
--ultrathink : enable ultra thinking mode
--plan : create a plan for the conversation
```
