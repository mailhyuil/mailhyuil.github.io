# ai base AI 탐지

AI에 의해서 생성된 컨텐츠를 탐지하는 방법

## Turnitin

- AI 텍스트 패턴 분석 기반 탐지 기능을 자체 개발해서 사용함
- 기존 표절 검사와 함께 AI 생성 가능성을 가중치로 평가
- 주로 교육기관에서 학생 과제 조사용으로 쓰임

## GPTZero

- 텍스트가 사람이 썼는지 AI가 썼는지를 판단하는 데 초점을 맞춘 시스템
- perplexity (예측 어려움) → 문맥 패턴의 예측 가능성
- burstiness (변화 정도) → 문장 간 리듬/변화 세기
- 문장 구조 및 의미 연결성 분석
- 이 데이터를 종합해서 AI 생성 확률을 퍼센트로 보여줌.
