# data graph

- 정점(vertex === node && vertices === nodes)과 간선(edge)로 구성
- 하나의 노드가 여러개의 노드와 연결: **인접행렬, 인접리스트, 2D Grid**
- 하나의 노드가 제한된 노드만 연결: **트리, 리스트** (그냥 변수로 연결)

## 무한대 (INF)

- 방문할 수 있는 경로가 없는 경우 무한으로 표현
- 자기 자신을 방문하는 경우 무한으로 표현
- 초기화 시킬 때 사용
- 그래프에서 경로의 길이나 가중치를 '비교'하여 경로를 결정
- 계산한 값과 기존의 값을 비교해서 더 '작을 때' 새로운 값으로 변경, 그래서 탐색하지 않은 경로를 무한대로 표현한다..

## 그래프의 예시

- 위키피디아 문서 링크 관계
- 소셜 네트워크 친구 관계
- 지도 경로
- 웹 페이지 링크 관계
- 파일 시스템 디렉토리 구조
- 네트워크 라우팅
- 컴퓨터 프로그램 의존성
- 작업 순서 관계
- 이벤트 순서 관계
- 작업 예약 관계
