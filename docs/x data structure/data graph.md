## 무한대 (INF)

> 방문할 수 있는 경로가 없는 경우 무한으로 표현
>
> > 자기 자신을 방문하는 경우 무한으로 표현
> >
> > > 초기화 시킬 때 사용
> > >
> > > > 그래프에서 경로의 길이나 가중치를 '비교'하여 경로를 결정
> > > >
> > > > > 계산한 값과 기존의 값을 비교해서 더 '작을 때' 새로운 값으로 변경, 그래서 탐색하지 않은 경로를 무한대로 표현한다..
