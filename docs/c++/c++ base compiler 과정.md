# c++ base compiler 과정

## 전처리

> .cpp, .h -> .i 파일 생성

```md
1. 모든 문자 해석
2. "\" 문자 해석
3. 전처리 토큰으로 분리
4. 전처리기 실행
5. 실행 문자 섹으로 변경
6. 인접한 문자열 합치기
```

## 컴파일

> .s(.asm) 파일 생성

```md
7. 해석 유닛 생성
8. 인스턴스 유닛 생성
```
