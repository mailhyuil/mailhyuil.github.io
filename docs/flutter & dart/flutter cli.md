# flutter base cli

```sh
flutter run
flutter doctor
flutter pub
flutter channel
flutter upgrade
```

## build

```sh
flutter build apk # android
flutter build appbundle # Android 릴리즈용 패키지

flutter build ios # ios

flutter build web # web
flutter build windows # windows
flutter build linux # linux
flutter build macos # macos

flutter build aot # 네이티브 코드로 컴파일 (Ahead-Of-Time)
```
