## Container() vs SizedBox()

```
Container # width와 height를 넣지 않으면, 최대 크기로 확장 (display: block)
SizedBox # width와 height 둘 중 하나라도 설정하지 않으면, 크기는 child의 크기에 알맞게 설정 (display: inline-block)
```
