# algorithm base stable sort vs unstable sort (안정정렬 vs 불안정정렬)

데이터에 여러 기준으로 정렬해야 할 때 매우 중요한 개념  
e.g. 나이를 기준으로 이미 정렬되어 있는 데이터를 이름순으로 다시 정렬할 때 불안정 정렬을 사용하면 기존 나이순서가 뒤바뀔 수 있음.

## Stable sort (안정정렬)

정렬 후에도 **같은 값을 가진 요소**들의 기존 순서가 유지됨.

## Unstable sort (불안정정렬)

정렬 후에 **같은 값을 가진 요소**들의 순서가 뒤바뀔 수 있음.
