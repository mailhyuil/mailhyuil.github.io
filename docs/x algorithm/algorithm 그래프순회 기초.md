# algorithm 그래프순회 기초

vertex를 visit, check, update 하는 것

## 그래프의 요소

```txt
graph: 그래프
vertex: 정점
edge: 간선 (가중치가 있을 수 있다.)
```

## DFS의 용도

1. 경로가 연결되어 있는지 확인
2. 연결 요소 세기
3. 트리 기반 문제 해결 (LCA, subtree 크기 등)
4. 백트래킹 (모든 경우 수 탐색)
5. 모든 경우를 다 탐색해야할 때 (완전탐색)

## BFS의 용도

1. 최단 경로 찾기 / 최소 비용 찾기 (가중치가 모두 1일 때)
2. 레벨별 처리 (그래프의 레벨별로 처리하는 것)
3. Flood Fill 종류 (일러스트의 Fill 처럼 연결된 모든 영역을 채우는 것) (connected region 탐색)
