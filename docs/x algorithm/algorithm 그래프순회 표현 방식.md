# algorithm 그래프순회 방식

## 인접리스트 (Adjacency List)

**node 간 연결**을 나타내는 리스트

```js
const graph = [
  [1], // 0번 노드
  [0, 2, 3], // 1번 노드
  [1, 4], // 2번 노드
  [1, 4], // 3번 노드
  [2, 3], // 4번 노드
];
```

## 인접행렬 (Adjacency Matrix)

**node 간 연결**을 나타내는 2D 배열

```js
const graph = [
  [0, 1, 0, 0], // 0번 노드
  [1, 0, 1, 1], // 1번 노드
  [0, 1, 0, 1], // 2번 노드
  [0, 1, 1, 0], // 3번 노드
];
```

## 2D Grid Map

**공간(맵)**을 나타내는 2D 배열

```js
const graph = [
  [1, 0, 1],
  [0, 1, 0],
  [1, 0, 1],
];
```
