# bun

> nodejs보다 3배 빠른 JS 런타임 (Rust 기반)
>
> > 자체 패키지 매니저, 번들러 빌드 도구, 테스트 러너, 개발 서버 내장

## command

```sh
# 초기화 (템플릿 생성)
bun init

# 패키지
bun install
bun add <package>
bun add <package> --dev
bun remove <package>

# 스크립트 실행
bun run <script>
bun run <script> --watch

# 빌드
bun build

# 개발 서버
bun dev

# 테스트
bun test

# 설정
bun setup

# npx
bunx
bun x
```
