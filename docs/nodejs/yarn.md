# yarn

## install

```sh
npm i -g yarn
# yarn classic 사용
yarn set version classic
# yarn berry 사용
yarn set version stable
```

## usage

```sh
yarn or yarn install # npm install
yarn add <package> # npm install <package>
yarn add -D <package> # npm -D install <package>
yarn remove <package> # npm uninstall <package>
yarn <script> # npm run <script>
yarn upgrade <package>
npx <command> --yarn # npx <command>
```
