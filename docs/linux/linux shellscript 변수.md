# 리눅스 셸 변수

```sh
$$ : 쉘의 프로세스 번호 (pid)
$0 : 쉘 스크립트 이름
$1~$9 : 명령줄 인수
$* : 모든 명령줄 인수 리스트
$# : 인수의 개수
$? : 최근 실행한 명령어의 종료 값

$HOME : 홈 디렉토리 경로
$USER : 현재 사용자
$HOSTNAME : 현재 호스트
$OSTYPE : 현재 운영체제 유형
$PWD : 현재 작업 디렉토리 경로

$PATH : 환경변수 디렉토리 목록*

$SHELL : 사용 중인 셸
$LANG : 현재 language 설정과 인코딩 타입
$RANDOM : 랜덤 숫자
$SECONDS : 셸 스크립트가 실행된 시간
```
