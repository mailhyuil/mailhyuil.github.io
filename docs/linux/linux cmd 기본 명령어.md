```
lscpu: CPU 정보 확인
lsblk: 블록 장치 정보 확인
lshw: 하드웨어 정보 확인

nmap: 네트워크 스캐닝 및 포트 검사

who: 현재 로그인한 사용자 목록 표시
w: 사용자 활동 및 현재 로그인한 사용자 목록 표시

uname: 시스템 정보 확인
uptime: 시스템 부하 및 실행 시간 확인
reboot: 시스템 재부팅
shutdown: 시스템 종료

systemd-analyze: 부팅 시간 및 부팅 프로세스 확인.

fsck: 파일 시스템 검사 및 복구를 수행합니다.

bg 및 fg: 작업을 백그라운드 또는 포그라운드에서 실행합니다.

nohup: 터미널을 종료해도 실행이 계속되는 프로세스를 실행합니다.

vmstat: 가상 메모리 통계 및 시스템 상태를 표시합니다.

rsync: 파일 및 디렉터리 간에 동기화하거나 복사합니다.

tcpdump: 네트워크 패킷을 캡처하고 분석합니다.

ngrep: 패킷 캡처 및 검색을 위한 명령줄 도구로, 특정 패턴이나 키워드를 찾을 때 유용합니다.
ssldump: SSL/TLS 트래픽을 캡처하고 표시하는 도구로, 암호화된 연결을 분석할 때 사용됩니다.
dstat: 다양한 시스템 통계를 실시간으로 표시하는 도구로, CPU, 메모리, 디스크, 네트워크 등의 성능을 모니터링합니다.
```
