# linux package manager apt - Debian

> 데비안 계열(debian, ubuntu..)의 패키지 매니저

```sh
apt update
apt install <package>
apt remove <package>
apt search <package>

# apt-get
apt-get install <package>
apt-get remove <package>
apt-get search <package>
apt-get update
apt-get upgrade

apt-mark hold <package> # 패키지 업그레이드 방지
apt-mark unhold <package> # 패키지 업그레이드 허용
```
