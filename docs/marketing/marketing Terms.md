# ad base Terms

```txt
직접매출: 광고를 클릭해서 바로 구매가 일어난 경우
간접매출: 광고를 클릭해서 바로 구매가 일어나지 않고, 나중에 구매가 일어난 경우

직접 ROAS(로아스): 직접매출 / 광고비용 * 100
간접 ROAS: 간접매출 / 광고비용 * 100

ROI (알오아이): 투자 대비 이익

MMP (Mobile Measurement Partner): 와이즈트래커, AppsFlyer(앱스플라이어), Adjust(애드저스트), Singular(싱귤러)

KPI (Key Performance Indicator): 핵심 성과 지표

Audience (타겟)
조건 설정 (광고주): "최근 3일 내에 우리 앱에서 '장바구니'에 담기 버튼을 누른 사람들을 타겟으로 잡아줘."
유저 식별 (매체사): 광고 매체는 우리가 보낸 데이터(픽셀, S2S 포스트백)를 뒤져서 해당 조건에 맞는 유저들을 **하나의 바구니(Audience)**에 담습니다.
광고 노출 (매칭): 그 바구니에 담긴 유저가 인스타그램이나 유튜브를 켜는 순간, 매체 서버가 "아, 이 사람은 아까 그 바구니에 있던 사람이네!"라고 판단하고 준비된 광고를 딱 보여줍니다.

CTR: 클릭률

CVR: 전환율

CPA: 액션당 비용

CPC: 클릭당 비용

CPM: 1,000회 노출당 비용.

LTV: 고객 생애 가치

CAC: 신규 고객 획득 비용

UTM: 캠페인 추적을 위한 쿼리스트링
utm_source: 유입 소스 (e.g. "google", "facebook" 등)
utm_medium: 매체 (e.g. "cpc", "display" 등)
utm_campaign: 캠페인 명 (e.g. "shoes_spring_sales_2026")
utm_content: 광고 소재 구분 (e.g. "some_link")
utm_term: 검색 키워드 (e.g. "신발 추천")

RTB: 실시간 입찰

어트리뷰션: 매출이 발생했을 때 이건 어떤 광고 덕분인가 판별하는 규칙 (First Touch, Last Touch, Lookback window)

포스트백: 마케팅 테크(MarTech) 분야에서는 광고를 통해 유입된 유저가 앱 설치나 구매 같은 '특정 행동'을 했을 때, 광고주(또는 우리 서비스)의 서버가 광고 매체(구글, 메타 등)의 서버에 "이 유저가 방금 전환을 일으켰어!"라고 알려주는 행위를 말합니다.
LLM 정제가 유리한 경우 (Signal Enhancement)
상담 로그 분석: 채팅 상담 내용(비정형)을 LLM으로 분석해 유저의 **구매 의도(High/Low Intent)**를 추출한 뒤, 이 등급을 포스트백으로 보낼 때.
이벤트 카테고리화: 서비스 내 수백 개의 클릭 이벤트를 일일이 하드코딩하기 힘들 때, LLM이 문맥을 보고 Lead, Wishlist, Search 등으로 자동 매핑해줄 때.
예측 데이터 생성: 유저의 행동 패턴을 보고 "이 유저는 30일 내 이탈할 확률이 높음"이라는 예측 라벨을 붙여 매체사에 보낼 때.
웹/앱 마케팅에서 사용자의 설치, 구매 등 특정 행동 데이터를 어트리뷰션 툴(MMP)이 광고 매체(애드 네트워크) 서버로 실시간 전송하는 서버 간(S2S) 데이터 교환 기술

피벗리포트: 수만, 수천만 개의 흩어져 있는 로우(Raw) 데이터를 사용자가 원하는 기준에 따라 '비틀어서(Pivot)' 요약하고 그룹화해 보여주는 보고서
통계, 집계 리포트, 그 데이터를 사용자가 원하는 대로 가로줄(행)과 세로줄(열)을 바꿔가며 입체적으로 보여주는 '형식'이 피벗리포트입니다.
```
