# db query Set Operators - UNION, INTERSECT, EXCEPT

## UNION

```sql
SELECT column1, column2 FROM table1
UNION
SELECT column1, column2 FROM table2;
```

## UNION ALL

```sql
SELECT column1, column2 FROM table1
UNION ALL
SELECT column1, column2 FROM table2;
```

## INTERSECT

```sql
SELECT column1, column2 FROM table1
INTERSECT
SELECT column1, column2 FROM table2;
```

## EXCEPT

```sql
SELECT column1, column2 FROM table1
EXCEPT
SELECT column1, column2 FROM table2;
```
