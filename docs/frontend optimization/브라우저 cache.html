브라우저 cache

cache-control을 사용하면 브라우저에서 캐싱을 할 수 있다.

같은 api 요청에 post 요청을 보내면 자동으로 캐시 버스팅이 된다.

Cache-Control외에도 Etag, Last-Modified, Expires, Pragma, Vary 등의 헤더가 있다면 브라우저는 캐시를 사용한다.
