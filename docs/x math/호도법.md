## 원주율이란?

> 3.14159265358979323846...
>
> > 원주와 지름의 비율
> >
> > > 원주 = 원의 둘레

```txt
원주:지름 = 𝝅
```

## 원주 구하는 공식

```txt
원주 = 2𝝅r
```

## 원의 넓이를 구하는 공식

```txt
S = r² \* 𝝅
```

## 1 radian이란?

> r 값과 부채꼴의 원주의 길이가 같을 때 === 1radian

```txt
1 radian = 180degree / 𝝅
```

## radian을 각도법으로 변환

```txt
𝝅 = 180 degree
2𝝅 = 360 degree
```
