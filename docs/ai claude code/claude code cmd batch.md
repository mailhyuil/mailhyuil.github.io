# claude code batch

- 코드베이스를 조사하여 작업을 5~30개 단위로 분해, git worktree를 사용하여 병렬로 실행
- 대량의 데이터를 처리해야 하는 경우
  1. 대규모 평가: 수천 개의 테스트 케이스를 효율적으로 처리합니다.
  2. 콘텐츠 조정: 사용자 생성 콘텐츠의 대량을 비동기적으로 분석합니다.
  3. 데이터 분석: 대규모 데이터셋에 대한 인사이트 또는 요약을 생성합니다.
  4. 대량 콘텐츠 생성: 다양한 목적(예: 제품 설명, 기사 요약)을 위해 대량의 텍스트를 생성합니다.
- 즉시 응답이 필요하지 않은 경우
- 비용 효율성을 최적화하려는 경우
- 대규모 평가 또는 분석을 실행하는 경우

```sh
/batch 모든 파일을 console.log에서 src/logger.ts로 교체해줘
```
