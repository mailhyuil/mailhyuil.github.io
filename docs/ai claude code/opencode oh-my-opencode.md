```sh

/start-work # prometheus의 계획을 이어받아 개발한다.
```

## sisyphus agent

```sh
/agents sisyphus

/ulw # 스스로 백그라운드에서 에이전트 팀을 스폰하여 작업이 끝날때까지 멈추지 않고 개발
```

## cmd

```sh
/ralph     # 완료될 때까지 멈추지 않는 지속 실행
/ralplan   # 합의를 만들며 반복 계획

/plan      # 계획 인터뷰 시작
/autopilot # 자율 실행 플로우
```
