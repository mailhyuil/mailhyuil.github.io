# ELK Elasticsearch vs Vector stores

> "스타벅스 콜드브루"를 검색하면,
>
> > Elasticsearch → "콜드브루"라는 단어가 포함된 상품을 빠르게 검색
> >
> > Vector Store → "커피와 비슷한 음료"까지 포함해서 추천

## elasticsearch

> 역색인 기반
>
> > 키워드 검색, 필터링, 정렬, 집계에 최적화
> >
> > > 정형 데이터에 빠름

## vector stores

> 벡터 임베딩 기반
>
> > 의미론적 검색, 유사도 검색에 최적화
> >
> > > 자연어 검색, 추천 시스템, 이미지 검색
