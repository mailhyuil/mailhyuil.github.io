# postgres operators

## 1. Comparison operators

```sql
=, <>, <, <=, >, >=

IS NULL, IS NOT NULL
-- the same as
ISNULL, NOTNULL
```

## 2. Logical operators

```sql
AND, OR, NOT
```

## 3. Arithmetic operators

```sql
+, -, *, /, %
```

## 4. Bitwise operators

```sql
&, |, #, ~, <<, >>
```

## 5. String operators

```sql
||

LIKE, ILIKE
SIMILAR TO, NOT SIMILAR TO
```

## 6. Array operators

```sql
=, <>, <, <=, >, >=, @>, <@, &&, ||, array[]
```

## 7. Range operators

```sql
=, <>, <, <=, >, >=, <<, >>
```

## 8. Network address operators

```sql
<<, <<=, >>, >>=, &<, &>, <<=
```

## 9. Geometric operators

```sql
<, <=, =, <>, >=, >
```
