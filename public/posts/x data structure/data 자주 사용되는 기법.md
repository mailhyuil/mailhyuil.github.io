## 콜렉션

> 메모리에 담긴 데이터들의 집합!
>
> > 메모리에 아무렇게나 데이터를 집어넣으면 한번에 처리하기 힘드니까 자료구조로 묶어두는 것
> >
> > > 모든 자료구조는 기본적으로 "배열", "레코드" 딱 두가지만 사용해서 구현된다.
> > >
> > > > 배열 : 데이터를 연속된 메모리 공간에 저장하는 것
> > > >
> > > > > 레코드 : 데이터의 주소값을 저장하는 것
