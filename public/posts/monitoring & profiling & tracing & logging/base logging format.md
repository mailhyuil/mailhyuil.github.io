# logging format

## json

```json
{
  "level": "info",
  "message": "Hello distributed log files!",
  "service": "user-service",
  "timestamp": "2021-07-01T12:34:56.789Z"
}
```

## logfmt

```txt
level=info message="Hello distributed log files!" service=user-service timestamp=2021-07-01T12:34:56.789Z
```
