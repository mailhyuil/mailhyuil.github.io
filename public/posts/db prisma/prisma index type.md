# prisma index type

# Hash

> 해시 인덱스는 키를 해시 함수에 넣어서 생성된 해시값을 기반으로 데이터를 저장하고 검색합니다.
>
> > 해시 인덱스는 해시 충돌이 발생할 수 있으므로 중복된 해시값을 처리하기 위한 메커니즘을 갖추고 있습니다.

# Gin (Generalized Inverted Index)

> Gin 인덱스는 역 인덱스 방식을 사용하여 여러 열의 데이터를 색인화합니다.
>
> > Gin 인덱스는 텍스트 검색, 배열 검색 등 여러 유형의 데이터를 효율적으로 처리할 수 있습니다.

# Gist (Generalized Search Tree)

> Gist 인덱스는 다양한 데이터 유형에 대해 사용될 수 있으며, 각 데이터 유형에 대해 특화된 키 추출 및 비교 함수를 사용합니다.
>
> > Gist 인덱스는 범위 검색, 공간 데이터 검색 등 다양한 유형의 검색을 지원합니다.

# SpGist (Space-Partitioned Generalized Search Tree)

> SpGist 인덱스는 공간 데이터의 색인화를 위해 사용되며, 다차원 공간 분할 방법을 기반으로 데이터를 저장하고 검색합니다.
>
> > SpGist 인덱스는 공간 데이터 검색에 효율적입니다.

# Brin (Block Range Index)

> Brin 인덱스는 블록 범위에 대한 정보를 사용하여 데이터를 저장하고 검색합니다.
>
> > 블록 범위는 연속된 데이터 블록을 나타내며, Brin 인덱스는 데이터의 정렬 순서를 기반으로 인덱싱합니다.
> >
> > > Brin 인덱스는 데이터가 정렬되어 있는 경우에 효율적입니다.
