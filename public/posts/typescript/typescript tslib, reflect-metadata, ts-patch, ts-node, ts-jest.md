# typescript tslib, reflect-metadata, ts-patch, ts-node, ts-jest

## tslib

> TypeScript 컴파일러가 생성하는 JavaScript 코드에서 공통적인 유틸리티 함수를 제공하는 라이브러리
>
> > 유틸리티 함수들이 들어있음

## reflect-metadata

> TypeScript의 데코레이터(decorator)에 대한 리플렉션(reflection) 지원을 추가하는 라이브러리

## ts-patch

> TypeScript의 동작을 변경하거나 기능을 확장하기 위한 패치(patch) 도구

## ts-node

> TypeScript 파일을 직접 실행할 수 있는 노드(Node.js) 실행기

## ts-jest

> TypeScript로 작성된 코드를 위한 Jest 테스트 프레임워크의 플러그인
