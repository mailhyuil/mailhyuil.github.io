# server RAID

> 백업을 하기 위해
>
> > 용량을 데이터 손실 없이 증설하기 위해
> >
> > > 속도, 안정성, 효율성, 가용성을 높이기 위해

## 미러링 (mirroring) vs 스트라이핑 (striping)

## RAID 0

> 두 개 이상의 디스크에 데이터를 무작위로 분산하여 저장하는 방식
>
> > IO 속도 향상
> >
> > > 데이터 손실 가능성 증가

## RAID 1

> 두 개 이상의 디스크에 동일한 데이터를 저장하는 방식
>
> > 데이터 손실 가능성 감소 가용성이 높고 데이터 복구가 쉬움
> >
> > > 용량이 50% 감소

## RAID 6

> 두 개 이상의 디스크에 데이터를 분산하여 저장하고, 패리티 정보를 추가하여 데이터 손실 가능성을 줄이는 방식
>
> > RAID 5와 비슷하지만, 패리티 정보를 2개를 저장하여 데이터 손실 가능성을 줄임
> >
> > > 약간의 공간을 희생하고, RAID 5보다 느림
