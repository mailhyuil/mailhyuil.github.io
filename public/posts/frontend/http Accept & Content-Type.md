# http Accept & Content-Type

> 데이터 타입 (MIME)을 지정하는 헤더

## Accept

> 브라우저 -> 서버
>
> > 브라우저가 서버에게 자신이 처리할 수 있는 데이터타입을 명시
> >
> > > 서버에서 웬만하면 이 데이터로 가공해서 보내달라고 요청하는 것

## Content-Type

> 현재 전송하는 데이터가 어떤 타입인지 명시
> 브라우저, 서버는 우선적으로 이 헤더를 확인한다.
>
> > 데이터를 파싱하고 문자열화 하는 우선적인 기준
> >
> > > 이 헤더가 존재하지 않으면 기본으로 text/plain으로 처리한다.
> > >
> > > > POST, PATCH, PUT 같이 body에 데이터를 담아서 보내는 경우에는 반드시 Content-Type을 명시해야 한다.
